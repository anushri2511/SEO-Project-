# ☕ SEO Project: Roastery Coffee House (Lucknow)

## 📌 Overview
This project is a complete SEO audit and strategy plan for Roastery Coffee House, a premium café in Lucknow.

## 🔍 Key Areas Covered
- Competitor Analysis
- Keyword Research (Ahrefs & Ubersuggest)
- SERP Analysis
- Technical SEO Audit
- Content Strategy & Blog Recommendations
- SEO Strategy Timeline
- Success Metrics

## 🎯 Objective
To improve organic visibility, keyword rankings, and customer engagement through a structured SEO strategy.

## 🛠 Tools Used
- Ahrefs
- Ubersuggest
- Google PageSpeed Insights

## 📈 Expected Results
- Increase in organic traffic (3–6 months)
- Better rankings for local keywords
- Improved engagement and visibility
